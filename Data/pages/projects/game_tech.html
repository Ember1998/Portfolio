Breeding Inventory Application




BIA is a specialized software platform designed to manage, analyze, and utilize plant breeding and research data and manipulate the data accordingly.
This project is built on .Net 8 with the integration of Azure service Application and Azure SQL Database.

It uses Phenome by providing researchers, breeders, and agricultural organizations with tools to store and organize genetic, phenotypic, and environmental information, track experiments and crop traits, and evaluate the performance of different plant varieties.
