Cotiviti is a company that enables healthcare organizations to deliver better care at lower cost through advanced technology and data analytics, helping to ensure the quality and sustainability of how healthcare is delivered in the United States.
Claim Process Engine (CPE)



CPE is a project that I have contributed to in the past, but I'm no longer involved in it. It was written in C#, and utilizes the .NET 6.0 framework with Akka.Net design pattern

CPE is a Claim processing engine developed for validating different kinds of edit logic for health insurance claims. In the project I have helped developing different kinds of edit logic for health insurance claims.

CPE is designed in microservice architectural where the services are derived into 3 service. They are Batch Portal, Editor Service and BatchProcessor.
Grading Site Application



Grading Site application is an internal application for COTIVITI which was developed for supporting training concepts for new hired as well as analysts and coders.

This Applcation was developed using .NET 7.0 framework with Angular 14.1.3 as front end development.

The CLEAN design pattern was implemented where the deployment was done in CICD pipeline having Bamboo as a build agent and octopus as deployment service.
