Website

Yellow express is a Courier based application project to build an innovative delivery platform that enables customer focused, fast and reliable delivery services for Yellow Express (https://yellowexpress.com.au/) which is an Australia based courier company which specializes in delivering and moving as their main services.

It improves customer and driver satisfaction by delivering a self-service e-commerce portal that enables users to be in control of their delivery experience.

This project is built on .Net 6 with the integration of serverless AWS lambda as Saas Application. It uses CQRS as design pattern and uses the concept of MediatR. It also uses AUTH0 as authentication and authorization services. React has been used for front end design.

It uses Locus and other third party services for managing the driver and location for delivery services.
